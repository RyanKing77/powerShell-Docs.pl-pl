# <a name="text-element-for-customitem-for-groupby-format"></a>Text, element — CustomItem, GroupBy (format)

Określa tekst, który jest dodawany do danych, który jest wyświetlany przez kontrolkę, takie jak etykiety, nawiasy kwadratowe, należy ująć danych i miejsca do magazynowania, aby wciąć danych. Ten element jest używany podczas definiowania sposobu wyświetlania nowej grupy obiektów.

— Element (w formacie) elementu ViewDefinitions (Format) widoku elementu (w formacie) GroupBy Element konfiguracji dla elementu formant niestandardowy widok (w formacie) dla elementu CustomEntries GroupBy (Format), formant niestandardowy elementu CustomEntry GroupBy (Format) Formant niestandardowy GroupBy (Format) elementu CustomItem CustomEntry GroupBy (Format) tekst elementu CustomItem dla grupowania (w formacie)

## <a name="syntax"></a>Składnia

```xml
<Text>TextToDisplay</Text>
```
