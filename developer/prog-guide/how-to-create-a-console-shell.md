# <a name="how-to-create-a-console-shell"></a><span data-ttu-id="c0ebe-102">Jak utworzyć powłokę konsoli</span><span class="sxs-lookup"><span data-stu-id="c0ebe-102">How to Create a Console Shell</span></span>

<span data-ttu-id="c0ebe-103">Program Windows PowerShell udostępnia narzędzia powłoki upewnij, nazywane także "Upewnij kit", który jest używany do tworzenia powłoki konsoli, który nie jest rozszerzalny.</span><span class="sxs-lookup"><span data-stu-id="c0ebe-103">Windows PowerShell provides a Make-Shell tool, also referred to as the "make-kit", that is used to create a console shell that is not extensible.</span></span> <span data-ttu-id="c0ebe-104">Utworzone za pomocą tego nowego narzędzia powłoki, nie można później rozszerzyć za pomocą przystawki programu Windows PowerShell.</span><span class="sxs-lookup"><span data-stu-id="c0ebe-104">Shells created with this new tool cannot be extended later through a Windows PowerShell snap-in.</span></span>

## <a name="syntax"></a><span data-ttu-id="c0ebe-105">Składnia</span><span class="sxs-lookup"><span data-stu-id="c0ebe-105">Syntax</span></span>

<span data-ttu-id="c0ebe-106">Poniżej przedstawiono składnię wykorzystywaną do uruchamiania powłoki upewnij z w obrębie pliku upewnij.</span><span class="sxs-lookup"><span data-stu-id="c0ebe-106">Here is the syntax used to run Make-Shell from within a make-file.</span></span>

```
make-shell
  -out n.exe
  -namespace ns
  [ -lib libdirectory1[,libdirectory2,..] ]
  [ -reference ca1.dll[,ca2.dll,...] ]
  [ -formatdata fd1.format.ps1xml[,fd2.format.ps1xml,...] ]
  [ -typedata td1.type.ps1xml[,td2.type.ps1xml,...] ]
  [ -source c1.cs [,c2.cs,...] ]
  [ -authorizationmanager authorizationManagerType ]
  [ -win32icon i.ico ]
  [ -initscript p.ps1 ]
  [ -builtinscript s1.ps1[,s2.ps1,...] ]
  [ -resource resourcefile.txt ]
  [ -cscflags cscFlags ]
  [ -? | -help ]
```
