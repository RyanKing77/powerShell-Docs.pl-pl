Poniżej przedstawiono przykłady kodu, aby uzyskać przykładowe polecenie cmdlet GetProc04. Jest to `Get-Process` przykładowe polecenia cmdlet opisane w [Dodawanie niekończące raportowania z błędów, do polecenia Cmdlet usługi](../cmdlet/adding-non-terminating-error-reporting-to-your-cmdlet.md).
