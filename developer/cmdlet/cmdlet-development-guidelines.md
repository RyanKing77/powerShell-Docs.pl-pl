# <a name="cmdlet-development-guidelines"></a>Wskazówki dotyczące projektowania poleceń cmdlet

Tematy w tej sekcji zawierają wskazówki dotyczące programowania, używane do wygenerowania poprawnie sformułowanym poleceń cmdlet. Wykorzystując typowe funkcje udostępniane przez środowisko uruchomieniowe programu Windows PowerShell i postępując zgodnie z tymi wytycznymi, mogą tworzyć niezawodne polecenia cmdlet przy minimalnym nakładzie pracy i zapewnić spójne środowisko użytkownika. Ponadto będzie zmniejszyć obciążenie testu, ponieważ często używane funkcje wymagają przetestowanie.
