# <a name="the-windows-powershell-ise-scripting-object-model"></a>Model obiektów skryptowych środowiska Windows PowerShell ISE
  Windows PowerShell® Integrated Scripting Environment (ISE) udostępnia jego funkcje za pośrednictwem odpowiedni skryptów model obiektów, który umożliwia pisanie skryptów programu Windows PowerShell, które manipulowania różnych aspektów visual i funkcjonalności programu Windows PowerShell ISE. Skryptów obiektów, ich właściwości i ich metod są zdefiniowane w tym celu.
